#  Car Accident Dataset - 5,000 videos
The dataset contains **5,000** videos capturing crashes occurring in **traffic accidents**, providing structured data for **detection** and **prediction** tasks. It is designed to support **traffic safety** research, focusing on **motor vehicle incidents** and **crash reports**. Specifically engineered to challenge **detection models** and enhance **traffic accident recognition** systems.- **[Get the data](https://unidata.pro/datasets/car-accident-video/?utm_source=github-sc&utm_medium=referral&utm_campaign=car-accident-video)**

## Dataset characteristics:
| Characteristic          | Data                                                                 |
|-------------------------|----------------------------------------------------------------------|
| Description             | Video contains: 3 seconds before the accident, the accident itself, and 3 seconds after the accident |
| Data type               | Video                                                                |
| Tasks                   | Detection, Prediction                                               |
| Total number of files   | 5,000                                                               |
